<h1>Responsive Emails</h1>
<p>
  A middleman project with some functions for help on development of responsive emails.
</p>
<p>
  Stack:
  <ul>
    <li>Ruby</li>
    <li>Middleman</li>
    <li>Sass</li>
  </ul>
</p>
<p>
  <h2>Setup:</h2>
  <pre><code>gem install middleman</code></pre>
  <p>
    After:
  </p>
  <pre><code>bundle install</code></pre>
</p>
